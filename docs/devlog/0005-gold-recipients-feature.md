# Добавление метки "Получатель золота" для персонажей

## 🎯 Проблема
Отсутствовала возможность отметить до 6 персонажей, которые являются получателями золота. Требовалось реализовать визуальное разделение на главной странице: сначала получатели золота, затем остальные персонажи. Также было неудобно добавлять несколько рейдов - приходилось открывать модальное окно для каждого рейда отдельно.

## ✅ Решение
- Добавлено поле `isGoldRecipient` в тип `Character`
- Реализована функция `toggleGoldRecipient()` в store для переключения статуса
- Чекбокс "Получатель золота" отображается **только в режиме редактирования**
- Реализовано разделение списка на два раздела: "Получатели золота" и "Остальные персонажи"
- Добавлена визуальная индикация (золотая рамка) для получателей золота
- Прогресс-бар золота отображается вверху страницы
- **Мульти-выбор рейдов** в модальном окне: можно выбрать сразу несколько рейдов и сложностей

## 📝 Изменённые файлы
1. `src/types/index.ts` - добавлено поле `isGoldRecipient` в интерфейс Character
2. `src/stores/characters.ts` - добавлена функция toggleGoldRecipient и поддержка isGoldRecipient в updateCharacter
3. `src/components/molecules/CharacterCard.vue` - чекбокс только в edit mode
4. `src/components/organisms/CharacterList.vue` - добавлены секции с разделением персонажей
5. `src/views/DashboardView.vue` - добавлен обработчик toggleGoldRecipient, кнопка добавления в хедер
6. `src/assets/styles/_variables.less` - добавлена CSS-переменная --color-gold
7. `src/components/molecules/GoldProgress.vue` - новый компонент для отображения прогресса золота
8. `src/components/organisms/RaidAssignmentModal.vue` - мульти-выбор рейдов

## 🚀 Как протестировать
### Получатели золота:
1. Открыть главную страницу
2. Проверить отображение прогресс-бара золота под заголовком
3. Перейти в режим редактирования (кнопка "Редактировать порядок")
4. Кликнуть по чекбоксу "Получатель золота" на карточке персонажа
5. Убедиться, что персонаж отображается в секции "Получатели золота" с золотой рамкой
6. Проверить ограничение - при 6 получателях чекбоксы у остальных персонажей блокируются

### Мульти-выбор рейдов:
1. Нажать "+ Рейд" на карточке персонажа
2. Выбрать несколько рейдов/сложностей (клик по ним)
3. Нажать кнопку "Добавить (N)" внизу модального окна
4. Все выбранные рейды добавятся сразу

## ⚙️ Важные детали
- Чекбокс "Получатель золота" показывается **только в режиме редактирования**
- **Новые персонажи автоматически становятся получателями золота**, если получателей меньше 6
- Лимит 6 получателей золота реализован через disabled-состояние чекбокса
- Таблица с золотом показывается **только у получателей золота**
- Прогресс-бар золота отображается вверху страницы (под заголовком "Персонажи")
- Кнопка "+ Добавить персонажа" перенесена в хедер страницы
- Модальное окно выбора рейдов поддерживает мульти-выбор с подсветкой выбранных элементов
- Данные сохраняются в localStorage вместе с остальными данными персонажей
- Drag-and-drop работает в пределах каждой секции независимо

## 🎉 Итог
Функционал "получателей золота" реализован: чекбокс для переключения статуса в режиме редактирования, ограничение до 6 персонажей, визуальное разделение на главной странице. Добавлен мульти-выбор рейдов для удобного назначения сразу нескольких рейдов.
