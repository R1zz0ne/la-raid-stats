# Режим редактирования на вкладке "Рейды"

## 🎯 Проблема
На вкладке "Библиотека рейдов" карточки всегда отображали кнопки "Редактировать"/"Удалить", что создавало визуальный шум и было неинтуитивно. Теги сложностей были отдельно, а информация о ГС и золоте — ниже с англоязычными лейблами.

## ✅ Решение
- Добавлен режим редактирования с переключателем в header
- Теги сложностей (DifficultyBadge) перенесены непосредственно в строку с информацией о ГС и золоте
- Убраны англоязычные лейблы сложностей — теперь только визуальные теги

## 📝 Изменённые файлы
1. `src/components/organisms/RaidCard.vue` - добавлен проп `editing`, теги DifficultyBadge перенесены в строку с деталями рейда с CSS Grid выравниванием, кнопки действий заменены на иконки ✎/✕
2. `src/views/RaidLibraryView.vue` - добавлено состояние `isEditingMode`, кнопка переключения режима, sticky header

## 🚀 Как протестировать
1. Перейти на вкладку "Рейды"
2. Убедиться, что карточки отображаются без кнопок действий
3. Нажать "Режим редактирования" — появятся кнопки
4. Нажать повторно — кнопки скроются

## ⚙️ Важные детали
- Паттерн идентичен DashboardView (isEditingMode + sticky header)
- Кнопки используют `stop` модификатор для предотвращения конфликтов

## 🎉 Итог
Режим редактирования рейдов реализован, интерфейс стал чище и интуитивнее.
