# Импорт персонажей из Оружейной (лостарк.рф)

## 🎯 Проблема
Пользователям приходилось вручную добавлять каждого персонажа вручную, указывая имя, класс и ГС. При наличии 10+ персонажей на аккаунте это занимало много времени и увеличивало вероятность ошибок.

## ✅ Решение
Добавлена возможность импорта персонажей из Оружейной (лостарк.рф). Пользователь вводит имя любого персонажа с аккаунта, система загружает список всех персонажей с серверами, классами и уровнями, после чего можно выбрать нужных для импорта.

### Основные изменения:
1. **Создан сервис парсинга** `src/services/lostarkApi.ts` - загружает HTML страницу персонажа и парсит список персонажей
2. **Создан composable** `useArmoryImport.ts` - управляет состоянием импорта (поиск, выбор, импорт)
3. **Создан компонент** `ArmoryCharacterItem.vue` - строка персонажа с чекбоксом
4. **Создан модальный компонент** `ImportFromArmoryModal.vue` - полный UI импорта
5. **Расширен characters store** - добавлен метод `importFromArmory()` для batch-операций
6. **Добавлена кнопка в DashboardView** - "Импорт из Оружейной"

### Особенности реализации:
- Автоматическое определение класса по иконке с Оружейной
- Группировка персонажей по серверам
- Подсветка уже существующих персонажей ("Уже есть")
- Обновление существующих персонажей (ГС + класс) при импорте
- Выбор/отмена выбора всех персонажей

## 📝 Изменённые файлы

3. `src/components/molecules/ArmoryCharacterItem.vue` - **новый** компонент строки персонажа
4. `src/components/organisms/ImportFromArmoryModal.vue` - **новый** модальное окно импорта
5. `src/stores/characters.ts` - добавлен метод `importFromArmory()`

## 🚀 Как протестировать
1. Запустить приложение (`npm run dev`)
2. На странице персонажей нажать кнопку "Импорт из Оружейной"
3. Ввести ник персонажа (например: `RizzOne`)
4. Нажать "Найти" - загрузится список персонажей аккаунта
5. Выбрать нужных персонажей (по умолчанию все выбраны)
6. Нажать "Импортировать" - персонажи добавятся в список
