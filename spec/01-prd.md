Контекст проекта:

Мы разрабатываем frontend-приложение для игроков Lost Ark. Это вспомогательная система для учета пройденных рейдов и количества полученного золота за прохождение рейдов.

Приложение должно позволять:
1. Видеть список рейдов (по 3 на каждого игрового персонажа)
2. Возможность отмечать рейды которые пройдены
3. Возможность настривать список рейдов для каждого игрового персонажа
4. Возможность настраивать и добавлять новые рейды (Название рейда, сложность, сколько дают обычного и лимитированного золота за прохождение)
5. Возможность менять цветовую тему страницы (светлая, темная)
6. Важно чтобы настройки хранились на клиенте/в браузере, бэкенд тут не предусматривается, чтобы при переоткрытии браузера настройки не терялись
7. Возможность выгрузить и загрузить свой конфиг с набором персонажей, настроенных для них рейдов и настройки самих рейдов.
